# LAYO HAIR

**Where Every Strand Tells a Story**

A mobile-first salon booking website for LAYO HAIR, featuring 12 stunning hairstyle galleries and an easy appointment booking system.

## Features

- 12 Hairstyle galleries with high-quality images
- 3-step booking wizard
- Mobile-first PWA design
- Offline support

## Styles Available

Braids | Cornrows | Twists & Locs
-------|----------|---------------
Short Boho Braids | Fulani Braids | Island Twists
Knotless Braids | Double Cornrows | Invisible Locs
Lemonade Braids | Half Cornrows | Bantu Knots
French Curls | Stitch Braids | All Back Cornrows

## Getting Started

1. Clone the repository
2. Open `v2/index.html` with Live Server
3. Or open `layoweb.html` for the original version

## Tech Stack

- HTML5, CSS3, JavaScript
- Progressive Web App (PWA)
- CSS Custom Properties
